## 3. 使用方法

### 3.1 环境准备
#### 3.1.1 系统要求
- 操作系统：Windows 10+/macOS 10.15+/Linux (Ubuntu 18.04+)
- Python版本：3.13+
- 内存：至少4GB RAM
- 磁盘空间：至少500MB可用空间

#### 3.1.2 依赖安装
```bash
# 克隆项目
git clone {仓库地址}

# 进入项目目录
cd {项目目录}

# 创建虚拟环境（可选但推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```
