## Day 0 事项模板

### 事项1：竞品调研与定位分析

**事项描述模板**：
```
1. 使用搜索工具调研 [选定平台] 上 [选定领域] 的 3-5 个头部/对标账号
2. 分析它们的：核心选题方向、视觉风格（封面/头像）、爆款标题特征、用户互动特点
3. 结合用户需求，产出《账号定位分析报告》，包含：
   - 账号名称建议（3个备选）
   - 简介文案（3个版本）
   - 差异化定位建议
```

**执行要点**：
- 使用搜索工具查找" [平台] [领域] 头部账号"、" [平台] [领域] 博主推荐"
- 分析头部账号的共性特征和差异化点
- 账号命名要包含关键词且易记，避免生僻字
- 简介文案要明确"我是谁+做什么+提供什么价值"

**竞品调研维度**：
- 选题方向：垂直领域、内容形式（图文/视频/长文）
- 视觉风格：配色方案、字体选择、封面图风格
- 爆款标题：关键词使用、情绪触发、数字使用、提问句式
- 互动特点：评论区互动方式、话题引导、福利活动

**《账号定位分析报告》示例**：

```markdown
# 账号定位分析报告

## 竞品分析结果
调研对象：小红书职场领域头部账号 5 个

### 共性特征
- 选题方向：职场技能、沟通技巧、情绪管理、职业规划
- 视觉风格：简约清新、黄色/蓝色为主色调
- 标题特征：数字开头 + 解决痛点 + 实用工具
- 互动方式：评论区问答、私信咨询、社群引流

### 差异化建议
- 定位差异：聚焦"95后职场新人"特定群体
- 内容差异：增加真实案例拆解 + 模板工具
- 风格差异：采用朋友式语气，避免说教感

## 职号名称建议（3个备选）
1. 95后职场进化论
2. 新人职场第一课
3. 职场通关指南

## 简介文案（3个版本）

**版本1 - 价值型**：
95后职场人 | 分享3年从新人到主管的实战经验
提供：沟通技巧/向上管理/职业规划/效率工具
让每一步成长都有迹可循

**版本2 - 情感型**：
我是xx，一个摸爬滚打的95后职场人
记录职场路上的坑与成长
希望我的经历能给你一点启发
一起打怪升级，职场不迷茫

**版本3 - 简洁型**：
95后职场博主 | 真实案例+实用模板
专注新人职场成长，每周更新2-3篇
关注我，职场少走弯路

## 差异化定位
核心差异化：真实案例拆解 + 可执行模板工具
- 每篇内容包含1个真实职场案例
- 提供可直接使用的工具模板（如沟通话术模板、OKR模板等）
- 建立读者互动机制：每周选取1个粉丝问题进行解答
```

## 持续运营循环机制

> 重要提示：首周完成后，必须进入周复盘→下周规划→下周执行的循环，不要停止。

### 循环流程图

```
┌─────────────┐
│   首周执行   │
│ (阶段1-4)   │
└──────┬──────┘
       │
       v
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   周复盘     │ ───> │  下周规划    │ ───> │  下周执行    │
│ (阶段5-步骤1)│      │ (阶段5-步骤2)│      │ (阶段5-步骤3)│
└──────┬──────┘      └──────┬──────┘      └──────┬──────┘
       │                    │                    │
       └────────────────────┴────────────────────┘
                          │
                          v
                    （持续循环）
```

### 循环触发机制

**自动触发场景**：
1. 首周事项全部完成后，自动进入第一轮周复盘
2. 每周日/周一，自动输出周复盘和下周规划草案
3. 用户主动提到"周总结"、"本周数据"、"下周计划"等关键词时
4. 用户反馈数据表现或运营调整需求时

### 每周优化方向

**根据数据优化选题**：
- 表现好的内容：延展相关话题，深挖垂直领域
- 表现差的内容：分析原因，调整切入角度或表达方式
- 互动高的内容：增加话题引导，鼓励用户参与讨论

**根据反馈优化风格**：
- 用户点赞多的内容方向：增加此类内容的产出比重
- 用户评论关注的点：在后续内容中加强回应
- 负面反馈或争议：分析是否需要调整表达方式或选题方向

**根据效率优化流程**：
- 创作时间紧张：准备内容模板、素材库
- 发布环节耗时长：提前准备好标签、文案
- 互动跟进不及时：制定互动时间安排

### 长期运营策略

**第1-2周：磨合期**
- 重点是测试选题方向和内容风格
- 尝试不同类型的内容，观察数据反馈
- 与用户建立初步互动和信任

**第3-4周：稳定期**
- 根据前期数据确定稳定的内容方向
- 形成固定的发布频率和风格调性
- 建立稳定的选题策划和创作流程

**第5-8周：增长期**
- 根据数据优化爆款内容的复现
- 尝试新的内容形式或互动方式
- 考虑跨平台引流或合作推广

**第9周以后：迭代期**
- 总结前期运营经验，形成运营SOP
- 持续优化内容质量和用户互动
- 根据平台算法变化调整运营策略

|------|---------|---------|---------|------|
| Day 1 | [标题] | [内容概述] | [阅读量目标] | [标签] |
| Day 3 | [标题] | [内容概述] | [阅读量目标] | [标签] |
| Day 5 | [标题] | [内容概述] | [阅读量目标] | [标签] |

## 发布时间规划

- 发布时间：[具体时间段，如18:00-20:00]
